Штрихи к проекту

Основной целью проекта было создание универсальной системы для увеличения информативности объявлений.

Приложение использует изображение на информационном плакате для того, чтобы инициализировать данные и дополнить их, используя камеру мобильного телефона.

Для создания рекламы с поддержкой Ugmi требуется расположить Ugmi-метки в углах плаката, без изменения самого плаката.

Ugmi считывает инфоромацию с плаката и метки, после чего, проецирует дополненное изображение на плакат в реальном времени.
