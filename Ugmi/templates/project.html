Штрихи к проекту

Основной целью проекта было создание универсальной системы для увеличения информативности объявлений.

Приложение использует изображение на информационном плакате для того, чтобы инициализировать данные и дополнить их, используя камеру мобильного телефона.

Для создания рекламы с поддержкой Ugmi требуется расположить Ugmi-метку на вашем изображении.

Ugmi извлекает информации из Ugmi-метки, после чего дополняет информацию об объекте в реальном времени.
